Le site web [World Writing Systems](http://www.worldswritingsystems.org), en plus d'être un travail de recherche très intéressant sur l'histoire de l'écriture, est une belle source d'inspiration visuelle pour des éléments de jeu ou vos univers de jeux de rôle.
Ce site Web présente un glyphe pour chaque système d’écriture du monde. Il s’agit de la première étape du projet Missing Scripts, une initiative à long terme visant à identifier les systèmes d’écriture non encore codés dans la norme Unicode. À ce jour, il reste 146 scripts non encore codés en Unicode.

Le projet Missing Scripts est un travail collaboratif de :

- [Atelier National de Recherche Typographique (ANRT), Nancy, France](http://anrt-nancy.fr/en/)
- [Institut Designlabor Gutenberg (IDG), Hochschule Mainz, Germany](http://www.designlabor-gutenberg.de)
- [Script Encoding Initiative (SEI), Department of Linguistics, UC Berkeley, USA](http://www.linguistics.berkeley.edu/sei/)
